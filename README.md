## cryptoCLI
A Command Line Tool to interact with an API and fetch cryptocurrencies data to work with.

### Setting Up Development Environment
```
npm install   
npm link
```

### Commands - 
##### 1. Version :  
	> cryptocli -V

##### 2. Help :
	> cryptocli -h
	
##### 3. Manage API Key :
	> cryptocli key
	
#####	i) Set API Key :
	> cryptocli key set
	
#####	ii) Show API Key :
	> cryptocli key show
	
#####	iii) Remove API key :
	> cryptocli key remove
	
##### 4. Check Price :
	> cryptocli key check --coin=BTC --curr=INR
